# AI PDF Renamer

AI（Google Gemini）がPDFの内容を読み取り、適切なファイル名に自動リネームするデスクトップアプリです。

## 機能
- PDFの内容をAI（Google Gemini）が解析し、日付・文書名を自動判定
- リネーム形式: YYYYMMDD_文書名_末尾ラベル.pdf
- フォルダ一括処理対応
- 高速モード（先頭・末尾ページのみ送信）/ 全文モード

## ダウンロード
[最新版をダウンロード](https://github.com/Just2enough/AI_PDF_Renamer/releases/latest)

## 動作要件
- Windows 11（64bit）
- Google Gemini APIキーが必要
