# AI PDF Renamer

AI（Google Gemini / Anthropic Claude）がPDFの内容を読み取り、適切なファイル名に自動リネームするデスクトップアプリです。

## 機能
- PDFの内容をAI（Google Gemini / Anthropic Claude）が解析し、日付・文書名を自動判定
- リネーム形式: YYMMDD_文書名_末尾ラベル.pdf（日付形式は YYMMDD / YYYYMMDD を選択可能）
- フォルダ一括処理対応
- ファイルのドラッグ＆ドロップに対応
- 高速モード（先頭・末尾ページのみ送信）/ 全文モード
- 自動アップデート（新バージョンを起動時に検知し、ワンクリックで更新）

## ダウンロード
[最新版をダウンロード](https://github.com/Just2enough/AI_PDF_Renamer/releases/latest)

> v1.6.0 以降は自動アップデートに対応しています。インストール後は、新しいバージョンが出るとアプリ内で通知され、その場で更新できます。

## 動作要件
- Windows 11（64bit）
- Google Gemini または Anthropic Claude の API キーが必要
- インターネット接続環境が必要
